# New Year's Concert Archive

An interactive web archive of programmes from the Vienna Philharmonic's Neujahrskonzert (New Year's Concert). Explore decades of concert history, discover pieces and composers, and trace the evolution of this beloved annual tradition through an intuitive interface.

- Search functionality for pieces and composers
- Detailed programme information with links to recordings
- Statistics and historical data about conductors and performances
